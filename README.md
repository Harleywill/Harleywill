A comprehensive e-commerce solution with advanced features for ventilation product sales.

- **Backend:** Node.js/Express with JWT authentication
- **Database:** MongoDB with real-time inventory sync

**Tech:** React • Redux • Node.js • Express • MongoDB • Stripe • Google Analytics
